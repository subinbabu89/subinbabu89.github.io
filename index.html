Merch Audit

Special front and back-end application for performing audit in-store

Provide functionality for representatives to perform a merchant audit.As part of the audit, the representatives will complete a questionnaire and upload a picture taken from the device camera. This content will then be associated with the store’s ID and sent in a secure manner to a backend infrastructure. Once data has been sent to backend, the tablet must wipe clean all images taken as part of the audit process.
The back end infrastructure will expose all audit data sent from stores using a secure URL. Summary & detail views will be available to browse and review. Data and reports will be available as soon as the stores send information to the server.

  • Date: Mar 2012 - Dec 2012
  • Client: Bestbuy

Honey Encryption

Python Implementation of the encryption algorithm

Honey Encryption aims to produce a valid output for an invalid key. Currently attackers rely on brute-force applications which decrypts data by guessing the correct key. However, a cipher text that is honey-encrypted will yield valid-looking output messages. Thus an attacker cannot gain any information by use of such brute-force tactic.
After the leakage of many user passwords from various web sites in the past couple of years, researchers have learned that a significant number of people use weak, easily-predictable, and repeated passwords. Hence, no matter how secure the encryption algorithm maybe, such passwords were bound to be discovered by simple decryption attacks.

  • Date: Jan 2016 - May 2016
  • Client: Information Security Project
